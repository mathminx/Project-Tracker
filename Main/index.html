Instructions

Project Tracker displays all of your projects in a table. Each of your projects must have a name, a type, and a due date.

Add a project:

  1. Click the "Open Modal" button below.
  2. Input the name of the project.
  3. Select the project type from the drop-down menu.
  4. Select the due date using the interactive calendar.
  5. Click the "Submit" button.

Your project is added to the table!

To delete a project, click the "Delete" button net to the project you wish to delete.
